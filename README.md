# Dataset

- /data 디렉토리 참고

- keyword1 파일과 keyword2 파일의 항목을 조합하여 검색함.
따라서, 키워드 구성이 좋을수록 양질의 데이터를 얻을 수 있을 것으로 보임.

- 디렉토리 구조

```

```
